# *bob2020*

Example code and improvements automated via Criterion

## Quick start

Build and run benchmarks

```bash
$ cargo bench
```

Install criterion compare

```bash
$ cargo install critcmp
```

Summarise benchmark results

```bash
$ critcmp
```
